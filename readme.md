# kmh-to-mph [![Build Status](https://travis-ci.org/gillstrom/kmh-to-mph.svg?branch=master)](https://travis-ci.org/gillstrom/kmh-to-mph)

> Convert km/h to mph.


## Install

```
$ npm install --save kmh-to-mph
```


## Usage

```js
const kmhToMph = require('kmh-to-mph');

kmhToMph(50);
//=> 31.0685596
```
